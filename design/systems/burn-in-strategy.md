# Burn-In Strategy

Because the display runs continuously on a TV panel, static bright
elements should not remain in the exact same location on every screen.

Strategies used:

• Vary header layout across screens
• Move logo position (left/right)
• Move clock position depending on screen
• Slightly vary panel layout between screens

Elements that should remain fixed:

• Alert Bug (top right)

Known Issues To Address Later
-----------------------------
• Determine if additional minor UI shifts are needed
• Evaluate long-term panel wear patterns
