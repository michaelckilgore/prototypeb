# Sugar Hill Weather Network

## Forecast screen direction currently approved

- The expanded forecast subscreen should stay visually close to the approved blue broadcast header treatment.
- The top header should use the shared Sugar Hill Weather Network logo and the angled blue title band.
- The little white sparkle/twinkle near the right side of the header is desired.
- The clock should live in the top header on subscreens.
- The alert box should also be supported on subscreens in the upper-right area.
- The expanded forecast title can change by screen, but the overall header treatment should stay consistent across subscreens.
- The expanded forecast body should use four forecast periods stacked top to bottom.
- The 7-day strip should be shallower than earlier versions.
- The 7-day icons should be larger.
- The right-side current conditions box is not desired on the expanded forecast screen.
- Current conditions should live in the lower information bar on the forecast screen.
- SVG and PNG versions of the shared header/logo assets should both be kept for flexibility.
